Blog Design System

One system,
every post.

Color, type, spacing and components for a plain HTML & CSS blog — built from a warm clay-and-cream palette, an AAA-contrast neutral scale for light and dark reading, an Inter type scale, and a 4px spacing rhythm.

01 — Color

Brand + neutrals

Three brand colors carry the identity. A twelve-step AAA-contrast gray scale — tuned separately for light and dark backgrounds — handles everything structural: text, borders, surfaces.
